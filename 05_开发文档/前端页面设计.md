# 前端页面设计 - 贷款体检器v1.0

**项目名称**：企业贷款体质改造系统  
**产品名称**：贷款体检器 v1.0  
**技术栈**：React 18 + TypeScript + Ant Design + Tailwind CSS  
**更新日期**：2024-12-07

---

## 一、设计概述

### 1.1 设计原则
- **简洁直观**：界面简洁，操作流程清晰
- **响应式设计**：适配桌面端和移动端
- **渐进式引导**：分步骤引导用户完成诊断
- **数据可视化**：用图表展示诊断结果
- **快速响应**：加载反馈、操作反馈及时

### 1.2 设计风格
- **色彩**：
  - 主色：`#1890ff`（蓝色，专业、可信赖）
  - 辅色：`#52c41a`（绿色，成功）、`#faad14`（橙色，警告）
  - 背景：`#f0f2f5`（浅灰）
  
- **字体**：
  - 中文：思源黑体 / PingFang SC
  - 英文/数字：Roboto / SF Pro
  - 正文：14px，标题：16-24px

- **间距**：
  - 组件间距：16px、24px
  - 页面边距：24px（桌面）、16px（移动）

### 3.3 诊断问卷页

**页面目标**：引导用户快速完成15道题，体验流畅

**布局设计**：

```
┌─────────────────────────────────────────────────┐
│ Progress Bar                                     │
│ ████████████░░░░░░░░  8/15                      │
└─────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────┐
│ 问题卡片                                         │
│                                                  │
│  Q8. 您企业的年开票额是多少？                    │
│                                                  │
│  ┌─────────────────────────────────┐           │
│  │ 2400          万元               │           │
│  └─────────────────────────────────┘           │
│                                                  │
│  💡 提示：开票额越高，可贷额度越大               │
│                                                  │
│  [上一题]              [下一题]                  │
└─────────────────────────────────────────────────┘
```

**问题类型**：
1. **文本输入**：企业名称、年营收等
2. **数字输入**：年开票额、年纳税额等
3. **单选**：纳税等级（A/B/M/C）、征信状况等
4. **是否选择**：是否有房产、是否高新企业等

**交互设计**：
- 进度条实时更新
- 输入框自动聚焦
- 支持键盘Enter跳转下一题
- 上一题/下一题按钮
- 保存草稿（自动保存到localStorage）
- 最后一题显示"提交问卷"

**提交流程**：
```
填写完成 → [提交问卷] 
         ↓
      加载动画
      "AI正在分析您的企业数据..."
         ↓
      生成报告（3-5秒）
         ↓
      跳转到报告页
```

### 9.3 核心代码示例

**API请求封装**：

```typescript
// services/api.ts
import axios from 'axios';

const api = axios.create({
  baseURL: import.meta.env.VITE_API_URL,
  timeout: 10000,
});

// 请求拦截
api.interceptors.request.use((config) => {
  const token = localStorage.getItem('token');
  if (token) {
    config.headers.Authorization = `Bearer ${token}`;
  }
  return config;
});

// 响应拦截
api.interceptors.response.use(
  (response) => response.data,
  (error) => {
    // 错误处理
    return Promise.reject(error);
  }
);

export default api;
```

**React Query使用**：

```typescript
// hooks/useProducts.ts
import { useQuery } from '@tanstack/react-query';
import api from '@/services/api';

export const useProducts = (filters) => {
  return useQuery({
    queryKey: ['products', filters],
    queryFn: () => api.get('/products', { params: filters }),
    staleTime: 5 * 60 * 1000,
  });
};
```
