## 一、核心原则（铁律）

### 1.1 我们是什么？

✅ **我们是**：
- 贷款体质诊断工具
- 融资方案建议者
- 银行/服务机构的对接方

❌ **我们不是**：
- 银行或金融机构
- 放贷方
- 担保方

### 1.2 一句话边界

> **我们只做「诊断 + 建议 + 对接」，不做「承诺 + 担保 + 代办」。**

## 二、绝对禁止的表述（红线）

### 2.1 承诺类（绝对不能说）

| ❌ 禁止说 | ✅ 可以说 |
|----------|----------|
| "保证能贷到XX万" | "根据您的情况，预计额度区间在XX-XX万" |
| "100%能批" | "符合条件的情况下，通过率较高" |
| "肯定没问题" | "我们会尽力协助，但最终以银行审批为准" |
| "包下款" | "我们帮您匹配最合适的产品，但结果由银行决定" |
| "不成功不收费"（如果后端有成本） | "体检免费，后续服务按实际情况协商" |

### 2.2 利率/费用类（绝对不能说）

| ❌ 禁止说 | ✅ 可以说 |
|----------|----------|
| "最低利率3%" | "部分产品利率区间在3%-8%，具体以审批为准" |
| "比银行便宜" | "我们帮您对比多家产品，找到更优方案" |
| "没有任何费用" | "体检和咨询免费，后续服务费用透明告知" |
| "固定利率X%" | "利率会根据您的资质浮动，区间大概是X%-X%" |

### 2.3 时效类（绝对不能说）

| ❌ 禁止说 | ✅ 可以说 |
|----------|----------|
| "当天放款" | "部分产品最快X个工作日，具体以银行为准" |
| "秒批" | "线上产品审批较快，一般X-X个工作日" |
| "保证X天内到账" | "正常流程X-X个工作日，我们会跟进进度" |

### 2.4 身份类（绝对不能说）

| ❌ 禁止说 | ✅ 可以说 |
|----------|----------|
| "我们是银行的" | "我们是贷款诊断服务商，帮您对接银行" |
| "我们是XX银行合作方" | "我们熟悉多家银行产品，可以帮您匹配" |
| "我代表银行" | "我帮您分析情况，推荐合适的银行产品" |

## 三、额度预测的正确表述

### 3.1 预测时必须包含的限定词

每次给客户说额度预测时，**必须加上以下任意一个限定**：

- "预计"
- "区间"
- "大概"
- "根据您提供的信息"
- "以银行审批为准"

### 3.2 标准话术模板

**场景1：告诉客户预估额度**

```
❌ 错误："您能贷100万。"

✅ 正确："根据您提供的开票和纳税数据，预计额度区间在80-120万，
         具体以银行审批为准。"
```

**场景2：客户问"到底能贷多少"**

```
✅ 标准回答：
"我先说结论：预计区间是XX-XX万。

这个数是怎么算的呢？
- 您的年开票额是XX万，按发票贷算法大概是XX万；
- 您的年纳税额是XX万，按税贷算法大概是XX万；
- 综合来看，区间在XX-XX万。

但最终额度由银行根据您的完整资料审批，
我们的预测是参考值，不是承诺。"
```

**场景3：客户追问"能不能保证"**

```
✅ 标准回答：
"说实话，我们没办法保证。

银行审批会看很多因素：征信、流水、行业、抵押物等等，
有些我们能提前判断，有些要提交后才知道。

我们能做的是：
1. 帮您匹配最合适的产品；
2. 把材料准备齐全；
3. 全程跟进进度。

但最终批不批、批多少，决定权在银行。"
```

## 四、体检报告中的标准免责声明

**每份体检报告结尾必须包含以下文字**：

```
---
【重要说明】

1. 本报告基于您提供的信息生成，仅供参考，不构成融资建议或承诺。
2. 报告中的额度、利率、周期均为预估值，最终以银行或合作机构审批为准。
3. 银行政策可能随时调整，报告内容可能与实际情况存在差异。
4. 您有权自主选择是否采纳本报告的建议，以及是否使用我们推荐的合作方。
5. 本报告仅供您本人使用，请勿转发或用于其他用途。

报告生成日期：YYYY-MM-DD
```

## 五、常见敏感场景处理

### 5.1 客户说"你们和XX助贷公司有什么区别"

```
✅ 标准回答：
"我们的定位不太一样。

普通助贷公司通常是：您问能不能贷，他说能就帮您办，不能就说没办法。

我们多做一步：
- 先帮您做体检，看清现在的贷款体质；
- 再分析：是直接能贷，还是需要调整，还是暂时不适合；
- 给您2-3条可选路径，您自己决定走哪条。

简单说，我们更像'贷款医生'，先诊断再开方，而不是直接卖药。"
```

### 5.2 客户说"我朋友找XX办只要X%手续费"

```
✅ 标准回答：
"每家的收费方式不一样，我不好评价别人。

我们这边的情况是：
- 体检和咨询免费；
- 如果您后续需要我们对接银行或合作方，费用会提前透明告知；
- 您可以对比之后再决定。

核心是：我们会告诉您所有费用明细，不会有隐藏收费。"
```

### 5.3 客户说"你们能不能帮我'做'数据"

```
✅ 标准回答：
"抱歉，这个我们做不了，也不建议您做。

银行审批会交叉验证，数据造假一旦被发现，
轻则拒贷进黑名单，重则涉及骗贷，后果很严重。

如果您现在数据不够，我们可以帮您分析：
- 有没有合规的方式提升（比如开票习惯调整）；
- 需要多长时间；
- 成本多少。

但造假这条路，我们不走。"
```

### 5.4 客户问"H业务是不是虚开发票"

```
✅ 标准回答：
"不是，完全不一样。

虚开发票是：没有真实交易，凭空开票，这是违法的。

H业务是：真实的贸易操作，有真实的货物、合同、物流、仓单，
我们叫'四流合一'。发票是基于真实交易开的，完全合规。

简单理解：我们是帮您'真的做生意'，
让您的开票额自然增长，而不是造假。

如果您还有顾虑，我们可以详细解释操作流程。"
```

## 六、合伙人话术培训要点

### 6.1 合伙人必须背熟的3句话

1. **开场定位**：
   > "我们是做贷款体检的，免费帮您看看现在能贷多少、怎么能贷更多。"

2. **边界说明**：
   > "我们只做诊断和建议，不承诺结果，最终以银行审批为准。"

3. **转介绍话术**：
   > "如果您觉得有需要，我可以帮您对接专业顾问，详细聊聊。"

### 6.2 合伙人绝对不能说的话

- ❌ "我保证能批"
- ❌ "我们有内部渠道"
- ❌ "交给我肯定没问题"
- ❌ "不成功不收钱"（如果后端有成本）
- ❌ "比银行直接办便宜"

### 6.3 遇到不确定问题怎么办

```
标准回答：
"这个问题我需要确认一下，我帮您问问专业顾问，稍后回复您。"

然后：
1. 截图问题发到内部群；
2. 等专业顾问回复后，再转述给客户；
3. 不要自己编答案。
```

## 七、违规处理

### 7.1 内部人员

| 违规等级 | 行为 | 处理 |
|---------|------|------|
| 一般 | 用词不规范但无承诺性质 | 口头提醒 + 培训 |
| 严重 | 对客户做出承诺性表述 | 书面警告 + 扣绩效 |
| 重大 | 承诺导致客户投诉/损失 | 终止合作 + 承担责任 |

### 7.2 合伙人

| 违规等级 | 行为 | 处理 |
|---------|------|------|
| 一般 | 用词不规范 | 提醒 + 重新培训 |
| 严重 | 对客户做出承诺 | 暂停合作 + 整改 |
| 重大 | 导致客户投诉/损失 | 终止合作 + 不结算佣金 |
