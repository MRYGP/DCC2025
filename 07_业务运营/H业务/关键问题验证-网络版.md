## 一、养户周期是否必须1年？

### ✅ 验证结论：不是必须1年，但主流产品需要9-18个月

| 产品类型 | 最低成立时间 | 最低开票时间 | 来源 |
|---------|-------------|-------------|------|
| **微众微业贷** | 企业18个月/个体户1年 | 连续纳税24个月 | 微众银行 |
| **发票贷（通用）** | 1-2年 | 第一张发票≥9个月前 | 助贷平台 |
| **税贷（通用）** | 最低1.5年，普遍2年 | 连续纳税18-24个月 | 行业标准 |
| **商户贷/流水贷** | **6个月** | 无开票要求，看流水 | 部分互联网银行 |

### 🔑 关键发现

**4-6个月能贷到钱吗？**
- ✅ **可以**，但产品有限，额度较小：
  - 微众银行微业贷：注册满6个月可申请，最高300万
  - 商户贷/流水贷：营业执照满6个月+支付流水
  - 科创助力贷：特批可做6个月（需高新/专精特新资质）
  
- ⚠️ **但主流发票贷/税贷不行**：
  - 发票贷：第一张发票至少9个月前
  - 税贷：普遍要求成立2年+连续纳税24个月

### 📊 产品门槛对照表

| 周期 | 可做产品 | 典型额度 | 备注 |
|-----|---------|---------|------|
| **6个月** | 商户贷、流水贷、微业贷 | 10-50万 | 额度小，看流水 |
| **9个月** | 部分发票贷（易企贷等） | 50-100万 | 需有9个月开票记录 |
| **12个月** | 多数发票贷 | 50-300万 | 需1年开票≥9个月 |
| **18-24个月** | 税贷、微众微业贷 | 100-500万 | 额度更大 |

## 二、断票会有什么影响？

### ✅ 验证结论：断票对发票贷影响很大，是"扣分大项"

| 指标 | 安全线 | 危险线 | 后果 |
|-----|-------|-------|------|
| **连续断票** | ≤1个月 | **≥3个月** | 通过率很低 |
| **近3个月** | 有开票 | **断票** | 基本拒绝 |
| **近1年断票总月数** | ≤3个月 | **≥4个月** | 影响额度 |
| **最大断票间隔** | ≤60天 | **≥90天** | 部分拒绝 |

### 🔑 关键规则
```
✅ 安全：近1年断票≤3个月，无连续断票
⚠️ 危险：近3个月断票 或 连续断票≥2个月
❌ 拒绝：连续断票3个月 或 近3个月全断票
```

### 💡 应对策略
- 如果断票严重，优先做**税贷**（税贷对断票不太敏感）
- 确保**近3个月必须有开票**
- 控制全年断票≤4个月

## 三、开票下滑多少会影响？

### ✅ 验证结论：同比下滑≤50%安全，≤70%可进件

| 指标 | 安全线 | 可进件 | 危险线 |
|-----|-------|-------|-------|
| **近1年同比下滑** | ≤50% | ≤70% | >70% |
| **近6个月环比下滑** | ≤50% | ≤70% | >70% |
| **连续2年累计下滑** | ≤50% | ≤60% | >60% |

### 🔑 关键规则
```
✅ 安全：同比/环比下滑≤50%
⚠️ 可进件：下滑50%-70%（产品选择重要）
❌ 困难：下滑>70%（需特殊方案）
```

## 四、额度算法差异

### ✅ 验证结论：税贷和票贷算法不同

| 产品类型 | 宣传额度 | 实操额度 | 单家银行 | 最高额度 |
|---------|---------|---------|---------|---------|
| **税贷** | 年纳税×30倍 | 年纳税×5-10倍 | 年纳税×3-8倍 | 500万/家 |
| **发票贷** | 年开票×30% | 年开票×10-20% | 年开票×2-10% | 300万/家 |

### 🔑 实际公式（网络验证）
```
税贷额度 = 年纳税额 × 5-10倍（优质客户最高30倍）
票贷额度 = 年开票额 × 10%-20%（多家叠加）
单家银行 = 年开票额 × 2%-10%

例：年开票500万
→ 单家银行：10-50万
→ 多家叠加：50-100万
→ 理论上限：150万
```

## 五、其他关键要求（网络汇总）

### 纳税等级
| 等级 | 说明 | 可做产品 |
|-----|------|---------|
| A/B级 | 最优 | 全部产品 |
| C/M级 | 部分可做 | 大部分产品 |
| D级 | 不建议 | 极少数产品 |

### 法人变更
| 变更时间 | 影响 |
|---------|------|
| <6个月 | 仅1-2个产品可做 |
| 6个月-1年 | 大部分产品可做 |
| >3年无变更 | 通过率更高 |

### 征信要求（通用）
```
✅ 当前无逾期
✅ 近6个月逾期≤2次
✅ 近12个月无M2（逾期≥2个月）
✅ 近24个月无M4
✅ 资产负债率≤70%（最好≤50%）
✅ 近6个月查询机构≤15次
```

## 七、总结：H业务养户周期建议

### 最低可行方案（6个月）
```
开票6个月 → 申请商户贷/流水贷 → 预计额度10-50万
优点：周期短
缺点：额度小、利率可能较高
```

### 标准方案（9-12个月）
```
开票9-12个月 → 申请发票贷 → 预计额度50-150万
优点：额度适中、利率合理
缺点：需要一定养户时间
```

### 最优方案（18-24个月）
```
开票18-24个月 → 发票贷+税贷同时做 → 预计额度200-500万
优点：额度最大、利率最低、可多家叠加
缺点：周期长
```
