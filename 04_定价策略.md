# H业务定价策略 v1.0

**更新日期**：2025-12-07  
**定价原则**：价值定价，从客户能接受的程度出发

---

## 一、定价逻辑

### 1.1 客户视角的价值计算

```
客户获得的价值：
┌─────────────────────────────────────────────────────────────┐
│  1. 新增贷款额度：X万（本来贷不到的钱）                       │
│  2. 利息节省：X万 × (民间利率 - 银行利率)                    │
│  3. 未来增额空间：开票基础打好，以后可持续申请                │
└─────────────────────────────────────────────────────────────┘

举例（贷款100万）：
- 如果去借民间：100万 × 15% = 15万/年利息
- 如果借银行发票贷：100万 × 5% = 5万/年利息
- 每年节省：10万
- 客户获得价值 = 100万额度 + 10万/年利息节省

客户愿意付出 = 价值的 10%-20%
```

### 1.2 定价公式

```
收费 = 目标贷款额度 × 费率

费率区间：10%-20%
├── 关系客户/首批客户：10%-12%（或成本价）
├── 普通客户：15%-18%
└── 复杂/加急客户：18%-20%
```

## 四、报价话术

### 4.1 报价时

```
"根据您的情况，我们帮您把开票做到X万，预计能贷到Y万。"
"整个周期大概需要3-6个月，投入大约Z万。"
"相当于您花Z万，换Y万的低息贷款额度。"
```

### 4.2 客户说"太贵了"

```
"我们来算一笔账：
- 如果您去借民间，100万一年利息就要15-20万
- 银行发票贷利率才5%左右，一年利息5万
- 光利息差一年就能省10万
- 而且开票基础打好后，以后额度还可以涨
- 这Z万是一次性投入，以后每年都在省钱"
```

### 4.3 客户说"别人更便宜"

```
"便宜有便宜的做法，但我们做的是真实贸易，四流合一，完全合规。"
"您可以问问他们具体怎么做的：
  - 有没有真实货物？
  - 有没有真实合同？
  - 有没有真实仓单？
  - 资金怎么走的？"
"我们做完之后，您的开票记录是真实的，以后任何银行都认。"
```

### 4.4 客户问"能保证吗"

```
"我们不承诺100%成功，因为最终审批权在银行。"
"但根据以往案例，只要按标准做到位，成功率是很高的。"
"我们收费也是按节点来的，不是一次性收完。"
"最后30%是贷款批下来才付的，这也是我们对自己的约束。"
```

### 4.5 贷款没成功怎么办

```
"前期收的是成本费用，已经产生了真实开票记录。"
"如果是银行原因（政策变化等），我们协助申请其他银行。"
"如果是客户原因（征信变化等），费用不退，但开票记录保留，您可以后面再申请。"
```

## 六、定价FAQ

**Q1：客户目标额度很大（比如500万），怎么定价？**

A：
- 单靠发票贷可能做不到500万
- 可能需要组合方案（发票贷+抵押贷）
- 大客户可以适当降低费率（10%-12%）
- 但前期先不做组合，只做单纯的发票贷改造

**Q2：客户只想贷30万，值得做吗？**

A：
- 30万按15%收费 = 4.5万
- 成本可能要10万左右
- ROI很低，不太划算
- 建议：前期不接低于50万目标的客户

**Q3：客户已有一定开票（比如200万），只需要补到300万，怎么定价？**

A：
- 按增量定价，不按总量
- 增量100万开票，预计增加X万额度
- 收费 = 增加的额度 × 费率

**Q4：客户急需用钱，能不能加急？**

A：
- H业务本身需要时间积累，无法真正加急
- 如果客户急，可能不适合H业务
- 或者建议：先申请现有的产品（如有），同时做H业务为未来增额
