# 银行贷款产品深度研究 · AI执行指令

**用途**：分发给多个AI并行研究，最后汇总  
**研究范围**：全国通用的企业贷款产品（地方性银行暂不研究）  
**输出要求**：统一格式，方便整合到产品总表

---

## 📋 通用说明（每个AI都要看）

### 研究目标

帮助「贷查查」建立银行产品知识库，用于：
1. 根据企业体质匹配合适的贷款产品
2. 预估可贷额度和利率区间
3. 告诉企业需要准备什么材料

### 银行范围（全国通用）

**优先研究**：
- 六大行：工商、农业、中国、建设、交通、邮储
- 股份制：招商、平安、中信、浦发、民生、兴业、光大、华夏、广发、浙商
- 互联网银行：微众、网商、新网、苏商、众邦

**暂不研究**：
- 地方城商行（长沙银行、北京银行等）
- 农商行、村镇银行
- 外资银行

### 输出格式要求

每个产品用以下Markdown表格格式输出：

```markdown
## [银行名称] - [产品名称]

| 字段 | 内容 |
|------|------|
| **产品类型** | 税贷/发票贷/流贷/抵押贷/信用贷 |
| **申请方式** | 线上/线下/线上+线下 |
| **目标客户** | 小微企业/个体户/一般纳税人/等 |
| **经营年限** | ≥X年 |
| **纳税等级** | A/B/C/M级 |
| **开票要求** | 近12个月≥X万 / 无要求 |
| **纳税要求** | 近12个月≥X万 / 无要求 |
| **额度范围** | X万 - X万 |
| **额度算法** | 年开票×30% / 年纳税×10倍 / 等 |
| **利率区间** | X% - X%（年化） |
| **期限** | X个月 - X个月 |
| **还款方式** | 随借随还/先息后本/等额本息 |
| **担保方式** | 纯信用/抵押/担保 |
| **征信要求** | 近2年无逾期/当前无逾期/等 |
| **禁入行业** | 房地产/娱乐/等 |
| **放款速度** | 最快X天 |
| **所需材料** | 营业执照、法人身份证、等 |
| **信息来源** | 官网/公开资料/等 |
| **可信度** | 高/中/低（标注哪些信息不确定） |

**备注**：[补充说明，如地区限制、特殊条件等]
```

### 重要提示

1. **标注不确定的信息**：如果某个字段网上查不到确切数据，写"待核实"并说明
2. **注明信息来源**：是官网、新闻、还是推测
3. **区分"官方说法"和"实际操作"**：比如官方说最低3.2%，实际大多数人拿到4.5%
4. **不要编造数据**：查不到就说查不到

## 你的研究范围

**产品类型**：税贷（基于企业纳税数据的信用贷款）

**必须研究的产品**：
1. 建设银行 - 云税贷
2. 工商银行 - 税务贷
3. 农业银行 - 纳税e贷
4. 中国银行 - 银税贷
5. 交通银行 - 税融通
6. 招商银行 - 闪电贷（税贷版）
7. 平安银行 - 税金贷
8. 微众银行 - 微业贷（税贷版）

**如有余力，补充研究**：
- 浦发银行、民生银行、中信银行的税贷产品
- 其他你能找到的全国通用税贷产品

## 重点关注

1. **纳税等级要求**：A级/B级/C级/M级分别能不能做？
2. **额度算法**：年纳税额的多少倍？有上限吗？
3. **零申报容忍度**：连续几个月零申报会被拒？
4. **票税下滑影响**：今年比去年下降多少会影响额度？

## 输出要求

1. 每个产品一个表格（用上面的模板）

# 📊 汇总整理指令（研究完成后用）

```
我有4份银行贷款产品研究报告，分别是：
1. 税贷类产品
2. 发票贷类产品
3. 经营贷/流贷类产品
4. 抵押贷+政策贷+特殊人群贷

请帮我：

1. **去重**：如果同一个产品在多份报告中出现，合并信息

2. **标注可信度**：
   - 高：多个来源一致
   - 中：单一来源但信息详细
   - 低：信息不完整或有矛盾

3. **生成产品总表**：
   - 按产品类型分组
   - 标注哪些字段需要进一步核实
   
4. **生成"待核实清单"**：
   - 哪些产品的额度算法不确定？
   - 哪些产品的准入条件有疑问？
   - 建议通过什么渠道核实？

5. **生成"产品推荐矩阵"**：
   - 企业体质A（优质）→ 推荐哪些产品？
   - 企业体质B（一般）→ 推荐哪些产品？
   - 企业体质C（较差）→ 推荐哪些产品？
```

## ⚠️ 注意事项

1. **AI研究的信息需要二次核实**：
   - 银行产品政策经常变
   - 网上信息可能过时
   - 建议后续让实习生打电话确认

2. **地区限制要标注清楚**：
   - 有些产品名义上全国通用，实际只在部分城市能做
   - 这种情况要在备注里说明

3. **利率区间要写实际区间**：
   - 官方宣传的"最低3.2%"往往拿不到
   - 尽量找到实际大多数人拿到的利率
