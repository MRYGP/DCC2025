## 使用说明

### 如何使用研究框架

1. **快速了解竞品**（1-2小时）：
   - 填写模块1/2（基本面+创始人）
   - 填写模块11（对标分析）
   - 输出一页纸总结

2. **深度研究竞品**（6-8小时）：
   - 完整填写12个模块
   - 实际体验产品
   - 收集用户评价
   - 输出完整报告

3. **定期更新**（1小时/季度）：
   - 重点更新模块6/8/12
   - 关注融资/产品/战略变化
   - 更新行动建议

### 研究优先级

按照以下顺序研究竞品：

```
Fundera（P0） → Lendio（P0） → 信易贷（P1） → 
Kabbage（P1） → 金蝶征信（P2） → 传统中介（P2）
```

## 关键产出

每个竞品研究应产出：
1. ✅ **可借鉴清单**：直接可抄的功能/策略
2. ⭐ **差异化机会**：他们不做/做不好的地方
3. ❌ **打不过清单**：不要硬刚的领域
4. 🎯 **行动建议**：P0级别的执行计划
