## 💰 商业模式详解

### 核心逻辑

| 角色 | 提供什么 | 得到什么 |
|------|---------|---------|
| **平台（我们）** | 工具、方法论、后端资源、风控规则 | 每单后端服务费的平台分成 |
| **合伙人** | 时间、人脉、执行 | 每单的合伙人分成 |
| **企业主** | 付费（仅为落地结果付费） | 标准化体检、方案、资金落地 |
| **后端服务方** | 资金/执行能力 | 服务费/利息差（我们帮他们获客） |

### 收入来源

```
企业主最终落地贷款/业务
        ↓
    后端服务费/返佣
        ↓
┌───────────────────────┐
│  平台分成  │  合伙人分成  │
│   (X%)    │    (Y%)     │
└───────────────────────┘
```

**关键点**：
- 前端体检和方案 **免费**
- 只从后端服务费里分成
- **合伙人赚得越多，我们才赚得越多**
- **后端服务方是我们的"隐性投资人"**——我们帮他们获客，他们给我们分成

## 🎭 不同场景的说法

### 内部版（给自己和核心成员）

> 我们的核心不是卖一个"贷查查 App"，而是做一个「企业贷款医生 + 兼职合伙人平台」。
>
> 产品只是工具，真正的商业模式在于：
> - 用这套工具和方法论，让大量普通人能够低门槛地上手做**合规、专业的融资中介**；
> - 他们帮企业主省心、省时间、防被坑；
> - 同时也帮自己赚到中介佣金；
> - 我们作为平台，在每一单的后端服务费/返佣里抽成。

### 合伙人版（招募兼职时用）

> 我这边做的，不是传统那种"贷款公司招业务员"。
> 我们在搭建一套「企业贷款体检 + 合伙人平台」。
>
> - 你不用自己懂所有银行政策，我们把产品做成了**体检工具**
> - 你不用自己找后端资金通道，我们已经筛过一批合作方
> - 你也不用背办公室、团队的成本
>
> 你赚的是：每一单成交后，后端服务费/返佣里的**合伙人分成**；
> 我赚的是：在合伙人分成之外，平台的那一部分。
> **你不赚钱，我也赚不到钱，这是一个互赢结构。**

### 企业主版（不强调合伙人网络）

> 我们做的是「企业资金体检 + 贷款方案设计」。
>
> 对您来说：
> - 先做一份**免费的贷款体检**，看清楚现在能贷多少、在哪些银行、用什么产品；
> - 然后我们会给一份 A/B/C 方案：
>   - A：不折腾，当前体质能做的
>   - B：轻微调整能多出的额度/节省的利息
>   - C：体质改造，中长期能达到的空间
>
> **前端体检和方案不收您钱，我们主要赚的是后端机构的服务费/返佣。**

### 后端合作方版（谈合作时用）

> 我们在做一个「企业贷款体检 + 合伙人获客平台」。
>
> 对你们来说：
> - 我们帮你们**筛选和预处理客户**——不是随便推个人过来，而是做完体检、资料齐全、匹配你们产品条件的企业
> - 我们有一批合伙人在前端获客，你们只需要接单、审批、放款
> - 分成方式很简单：成交后，从服务费/返佣里给我们平台分成
>
> **你们不用养获客团队，我们帮你们把客户送到门口。**

## 🔧 技术系统定位

### 贷查查产品 ≠ SaaS收费

| 传统做法 | 我们的做法 |
|---------|-----------|
| 开发产品 → 卖系统收费 | 开发产品 → **给合伙人用** → 从业务分成 |
| 收入来自软件费 | 收入来自每单后端服务费分成 |
| 客户是企业 | 客户是合伙人（企业主是最终用户） |

### 技术系统的作用

| 模块 | 功能 |
|------|------|
| **诊断引擎** | 输入7连问 → 输出体质评分 + 建议路径 + 匹配产品 |
| **产品匹配** | 根据企业体质自动匹配银行产品 |
| **报告生成** | 体检报告 + 方案对比（A/B/C） |
| **培训工具** | 话术边界 + SOP流程 |

### 技术开发优先级

| 优先级 | 功能 | 目的 |
|-------|------|------|
| **P0** | 诊断接口 `/diagnose/basic` | 能出体检结果 |
| **P0** | 产品数据结构 | 能匹配产品 |
| **P1** | 报告生成 | 能给客户看 |
| **P2** | 前端界面 | 合伙人自助使用 |
| **P3** | 合伙人后台 | 规模化管理 |

**核心原则**：先跑通人工流程，再用技术提效。

## 👥 角色分工

### 平台侧（我们）

| 角色 | 职责 |
|------|------|
| **业务负责人（松阳）** | 定规则、跑案例、搭体系、对接后端 |
| **技术负责人（吕天）** | 诊断引擎、产品匹配、报告生成 |
| **产品实习生** | 产品录入、案例整理、资料归档 |

### 合伙人侧（未来）

| 类型 | 特点 |
|------|------|
| 代账公司 | 有企业客户、懂财务 |
| 保险经纪 | 有企业主人脉 |
| 工程/贸易朋友 | 有行业资源 |

### 后端侧（我们的"隐性投资人"）

| 类型 | 提供什么 | 我们帮他们做什么 |
|------|---------|----------------|
| 银行直连 | 最终放款 | 筛选合格客户 |
| 助贷机构 | 帮跑银行流程 | 预处理资料 |
| 过桥/典当 | 短期资金周转 | 匹配需求 |
| H业务执行方 | 体质改造执行 | 筛选改造对象 |

## 🚦 发展阶段

| 阶段 | 目标 | 关键动作 |
|------|------|---------|
