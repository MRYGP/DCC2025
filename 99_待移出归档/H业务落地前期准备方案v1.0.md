## 🔬 三、任务1：银行产品深度调研（发票贷）

### 3.1 调研目标

搞清楚3家主要银行发票贷的**真实情况**，包括：
- 准入门槛（硬性 vs 软性）
- 额度算法（真实公式）
- 审批流程（线上 vs 线下）
- 隐性要求（社保？流水？其他贷款？）
- 最近变化（政策调整？）

### 3.2 可网络研究的内容 🔬

| 调研项 | 内容 | 预期来源 |
|-------|------|---------|
| 产品基本信息 | 产品名称、最高额度、利率区间、期限 | 银行官网、公开资料 |
| 官方准入条件 | 成立年限、开票要求、行业限制 | 银行官网、产品说明书 |
| 申请入口 | APP名称、扫码入口、线上流程 | 银行官网、APP截图 |
| 所需材料（官方版） | 营业执照、法人身份证、开票清单等 | 银行官网 |
| 竞品对比 | 各银行发票贷横向对比 | 网络整理 |

**我可以做的**：
- 搜索3家银行发票贷的最新产品信息
- 整理成标准化的产品文档
- 做横向对比表

### 3.3 需要你验证的内容 ✅

| 验证项 | 验证问题 | 验证方式 |
|-------|---------|---------|
| **额度真实性** | 年开票100万，实际能批多少？算法是否准确？ | 找有经验的人问 / 用真实企业扫码试 |
| **隐性门槛** | 是否看社保？是否看银行流水？是否看其他贷款？ | 找银行客户经理问 / 找有经验的助贷问 |
| **开票连续性要求** | 必须连续12个月？还是累计12个月？中间断了怎么办？ | 找银行客户经理问 |
| **行业限制细节** | 贸易公司能做吗？哪些行业不行？ | 找银行客户经理问 |
| **审批时效** | 线上审批真的能当天出额度吗？实际多久？ | 用真实企业扫码试 |
| **最近政策变化** | 最近3个月有没有收紧或放宽？ | 找有经验的人问 |
| **额度叠加** | 同时申请多家银行发票贷，会互相影响吗？ | 找有经验的人问 |

**你需要做的**：
1. **找1-2个有经验的助贷朋友**，问上面的问题
2. **找1个银行客户经理**（交行/平安/招行），问准入细节
3. **用1个真实企业扫码试一下**，看实际能出多少额度

```markdown
| 标准项 | 要求 | 原因 | 如何验证 |

| 企业征信 | 无重大逾期/诉讼 | 银行准入门槛 | 查企业征信报告 |
| 有真实业务 | 非空壳公司 | H业务需要依托真实经营 | 看流水/实地了解 |

- [ ] 娱乐/KTV/夜总会
- [ ] 采矿/煤炭
- [ ] 钢贸（部分银行限制）

## 二、软性标准（优先满足）

| 标准项 | 理想情况 | 可接受情况 | 不建议 |
|-------|---------|-----------|--------|
| 年营收 | ≥1000万 | 500-1000万 | <500万 |
| 现有开票额 | ≥100万/年 | 50-100万/年 | <50万/年 |
| 目标贷款额度 | 100-300万 | 50-100万 | <50万（ROI低） |
| 时间紧迫度 | 不急，可以等6个月 | 稍急，希望4个月 | 非常急（不适合H业务） |
| 配合度 | 高（愿意提供网银盾、配合开票） | 中 | 低（不建议做） |
```

```markdown
## 一、诊断流程

```
Step 1：信息收集（30分钟）
    ├── 基础信息：企业名称、成立时间、行业、法人信息
    ├── 开票信息：年开票额、开票连续性、主要开票内容
    ├── 贷款信息：现有贷款、利率、还款情况
    └── 需求信息：想贷多少、用来干什么、多久要用
    ↓
Step 2：资质核查（1-2天）
    ├── 查企业征信（需要客户配合）
    ├── 查法人征信（需要客户配合）
    ├── 查企业经营状态（企查查/天眼查）
    └── 查企业开票数据（电子税务局）
    ↓
Step 3：现状评估（1小时）
    ├── 计算当前可贷额度（按现有开票额算）
    ├── 分析差距（目标额度 vs 当前额度）
    └── 判断改造可行性
    ↓
Step 4：方案设计（2-3小时）
    ├── 设计改造目标（年开票做到多少）
    ├── 设计改造路径（每月开票多少、用什么品类）
    ├── 估算成本和周期
    └── 预测改造后可贷额度
    ↓
Step 5：出具诊断报告

```markdown
## 一、定价逻辑

### 1.1 客户视角的价值

```
客户获得的价值 = 新增贷款额度 + 利息节省 + 未来增额空间

举例：
- 新增贷款额度：100万
- 利息节省：100万 × (民间15% - 银行5%) = 10万/年
- 未来增额空间：开票基础打好后，可持续申请

客户愿意付出 = 价值的 10%-20%
```

### 1.2 定价公式

```
收费 = 目标贷款额度 × 费率

费率区间：10%-20%
- 关系客户/首批客户：10%-12%（或成本价）
- 普通客户：15%-18%
- 复杂/加急客户：18%-20%
```

## 四、定价FAQ

**Q：客户说太贵了怎么办？**

A：算账给他看：
- "您贷100万，我们收15万，相当于您花15万换100万的低息资金"
- "如果您去借民间，100万一年利息就要15-20万"
- "而且开票基础打好后，以后额度还可以涨，这15万是一次性的"

**Q：客户说别人更便宜怎么办？**

A：
- "便宜有便宜的做法，但我们做的是真实贸易，合规操作"
- "您可以问问他们是怎么做的，四个真实能不能保证"
- "我们做完之后您的开票记录是真实的，以后任何融资都能用"

**Q：贷款没成功怎么办？**

A：
- 前期收的是成本费用，已经产生了真实开票
- 如果是银行原因（政策变化等），我们协助申请其他银行
- 如果是客户原因（征信变化等），费用不退但开票记录保留
```
