# H业务话术库 v1.0

**更新日期**: 2025-12-08  
**适用对象**: 销售团队、客户经理、执行团队  
**用途**: 标准化沟通话术,提升沟通效率和签约成功率  
**版本**: v1.0

---

## 📌 话术使用原则

### 基本原则

1. **真诚为先**: 话术是辅助工具,不是背诵剧本
2. **灵活调整**: 根据客户特点和现场情况灵活调整
3. **价值导向**: 始终围绕"帮客户解决问题"展开
4. **合规表述**: 避免承诺性语言,使用"通常""一般""根据经验"

### 禁用表述

❌ **禁止**:
- "保证100%成功"
- "固定利率X%"
- "秒批""当天放款"
- "零风险""绝对安全"

✅ **推荐**:
- "根据以往案例,通过率约XX%"
- "利率通常在X%-X%区间"
- "审批时间一般X-X天"
- "风险可控,有完整预案"

## 🎯 场景1: 初次接触(筛选阶段)

### 1.1 开场白

**情境**: 第一次联系潜在客户

**目标**: 引起兴趣,获得进一步沟通机会

**话术**:
```
XX总您好,我是[公司名]的[姓名]。

我们是做企业融资服务的,最近帮了几个朋友企业
通过优化开票记录,从银行拿到了500-800万的低息贷款,
利率只有3%-4%。

您公司今年有没有融资需求?我想了解一下您的情况,
看看我们能不能帮到您。
```

**关键点**:
- 简洁自我介绍
- 给出具体案例(数字+效果)
- 询问需求,不要直接推销

---

### 1.2 需求探询

**话术1: 了解融资需求**
```
Q: 您公司今年有融资计划吗?
Q: 您觉得现在贷款额度够用吗?
Q: 如果能拿到更多贷款,利率更低,您有兴趣吗?
```

**话术2: 了解痛点**
```
Q: 您之前申请过银行贷款吗?
Q: 额度批下来多少?利率多少?
Q: 有没有遇到什么问题?比如额度不够,利率太高?
```

**话术3: 了解开票情况**
```
Q: 您公司年营收大概多少?
Q: 年开票额大概多少?
Q: 纳税等级是A级吗?
```

**应对技巧**:
- 多提问,少陈述
- 从客户痛点切入
- 记录关键信息

---

### 1.3 价值陈述(简版)

**话术**:
```
XX总,根据您的情况,我们可能能帮到您。

简单说,就是:
通过真实贸易操作,帮您把年开票额提升到XX万,
然后向银行申请发票贷,
预计能拿到XX-XX万额度,利率3%-4%。

整个周期大概6个月,
投入成本大约是最终贷款额的10%-15%。

您有兴趣了解更详细的方案吗?
```

**关键点**:
- 用最简单的语言解释业务逻辑
- 给出明确的数字(额度、利率、成本、周期)
- 留悬念,引导深入沟通

### 2.4 处理常见疑问

**Q1: "这个是合法的吗?"**

A1:
```
完全合法。我们做的是真实贸易:
- 有真实的货物(金属材料、钢材等)
- 有真实的合同和仓单
- 有真实的物流和发票
- 增值税正常申报缴纳,进项可以抵扣

这叫"四流合一",完全符合税法。
您开出去的票,都有真实的采购对应,
税务局查也没问题。
```

---

**Q2: "资金安全吗?会不会被卷走?"**

A2:
```
资金安全这块您完全放心:
1. 资金始终在您自己的对公账户
2. 我们无法转出,只能按约定流程操作
3. 资金不过夜,当天进当天出
4. 每笔操作提前通知您,您可以随时查看
5. 有完整的合同、发票、仓单做保障

您可以理解为:您把网银盾暂时给我们,
但钱还是在您账户里,我们只是帮您操作一下。
```

---

**Q3: "银行会不会发现?会不会影响贷款?"**

A3:
```
不会的。银行主要看三个东西:
1. 开票记录是否真实连续 - 我们保证真实
2. 企业经营是否正常 - 您本身经营正常
3. 资金流水是否异常 - 不过夜,不异常

而且,银行就是看开票额给额度的,
您开票多了,银行高兴还来不及。

如果银行回访,我们会提前给您准备话术,
就说"今年业务量增加,客户要求规范开票",
很正常的理由。
```

---

**Q4: "为什么不直接贷款,还要先做开票?"**

A4:
```
这是银行的规则决定的。

银行发票贷的逻辑是:
年开票额 × 30% = 可贷额度

您现在年开票XX万,所以只能贷XX万。
如果想贷更多,就必须先把开票做上去。

这个过程需要时间,所以需要6个月。
但一旦做上去,贷款就水到渠成了。
```

---

**Q5: "太贵了,能不能便宜点?"**

A5:
```
XX总,我理解您的顾虑。我们来算笔账:

【投入】XX万
【收益】新增贷款XX万,利率降X%,每年节省利息XX万

- 第一年:投入XX万,节省XX万,净收益XX万
- 第二年及以后:每年继续节省XX万

而且贷款额度是长期的,不是一次性的,
您可以循环使用好几年。

算下来,第一年就能回本,
后面都是纯收益。

您觉得这个投入值不值?
```

---

**Q6: "万一贷款批不下来怎么办?"**

A6:
```
这个担心可以理解。我们是这样的:

1. 前期我们会严格筛选,不符合条件的不接
2. 您的情况我们评估过,通过率在85%以上
3. 我们有成功案例,不是第一次做

但是,银行审批确实有不确定性,所以合同里写明:
- 如果是银行政策变化,我们协助申请其他银行
- 如果是您的征信突然出问题,费用不退(因为开票已产生)
- 如果是我们的原因,退还未执行部分费用

但根据经验,只要条件符合,基本都能批下来。
而且,就算这次没批,开票记录也保留了,
以后条件改善了,还可以再申请。
```

**异议: "太贵了,能不能打折?"**

**Step 1: 认同情绪**
```
XX总,我理解您的顾虑,
XX万确实不是小数目。
```

**Step 2: 价值对比**
```
但是我们来算笔账:
- 投入XX万
- 拿到XX万贷款
- 利率降X%,每年节省XX万
- 第一年就能回本,后面都是纯收益

相当于花XX万,买了一个每年产生XX万收益的资产,
您觉得值不值?
```

**Step 3: 成本拆解**
```
而且这XX万不是我们赚的,
实际成本构成是:
- H业务操作费:XX万(付给后端)
- 税费:XX万(实际缴纳)
- 我们服务费:XX万(协调、跟进、风控)

我们的利润空间其实很薄,
真心是想帮您解决问题,不是为了赚快钱。
```

**Step 4: 给出方案**
```
这样吧,我跟老板申请一下,
看能不能给您一个特别优惠:

[根据情况,可以给的优惠]:

这个我们确实控制不了,
已发生的成本不能退。

您觉得这样合理吗?
```

### 5.3 竞品对比异议

**异议: "XX公司说他们只要X%的费用"**

**应对策略**:

**Step 1: 不贬低竞品**
```
XX公司确实也在做这个业务,
我听说过,他们也有自己的优势。
```

**Step 2: 强调差异**
```
不过我们的差异在于:

【服务深度】
- 我们不只是做开票,还包括贷款申请全程协助
- 专人跟进,月度汇报,遇到问题及时解决

【风险控制】
- 我们有完整的四流合一材料,合规性强
- 如果遇到银行或税务问询,我们有标准应对方案

【成功率】
- 我们的通过率约85%,行业平均水平60-70%
- 我们会前期严格筛选,不行的不接,保证质量

【服务态度】
- 我们把客户当朋友,不是一锤子买卖
- 后续如果您有其他融资需求,我们也会持续帮您
```

**Step 3: 价值主张**
```
所以,虽然我们收费可能贵一点,
但是:
- 服务更全面
- 成功率更高
- 风险更小
- 后续支持更好

您是要便宜但不靠谱的,
还是要贵一点但更有保障的?
```

### 5.4 时间异议

**异议: "6个月太久了,能不能快点?"**

**应对策略**:

**Step 1: 解释原因**
```
XX总,我理解您着急。

但6个月是必须的,原因是:
1. 银行要看开票连续性,至少要6个月记录
2. 开票增长要循序渐进,不能突然暴增(风控)
3. 申请贷款本身也需要1个月

如果时间太短,银行会觉得不真实,
反而容易拒贷。
```

**Step 2: 给出替代方案**
```
但是,如果您确实很急,
我们可以考虑:

【方案1: 先申请其他产品】
- 比如流水贷、商户贷,3个月就能申请
- 额度虽然少一点(10-50万),但快
- 6个月后再申请发票贷,拿更高额度

【方案2: 缩短执行期】
- 从6个月压缩到4-5个月
- 但开票频率会更密集,风控风险增加
- 而且银行可能要求补充材料

您倾向哪个方案?
```

## 📋 附录

### 附录A: 常用金句

**价值主张类**:
- "我们不只是帮您贷款,是帮您建立长期融资能力"
- "投入X万,买一个每年产生XX万收益的资产"
- "花钱买时间,花时间省钱,您选哪个?"

**信任建立类**:
- "我们把客户当朋友,不是一锤子买卖"
- "您的成功,就是我们最好的广告"
- "我们敢承诺,是因为我们有底气"

**促成行动类**:
- "早一天启动,早一天拿到贷款"
- "机会不等人,名额有限"
- "今天不决策,明天可能后悔"

---

### 附录B: 禁忌话术

❌ **禁止说**:
1. "保证能贷到XX万" (不确定)
2. "利率固定X%" (银行审批)
3. "绝对没风险" (夸大)
4. "XX银行内部关系" (违规)
5. "不成功全额退款" (不现实)
6. "你不信就算了" (态度差)
7. "别人都签了,就你犹豫" (施压)

✅ **应该说**:
1. "根据经验,通常能贷到XX-XX万"
2. "利率一般在X%-X%区间"
3. "风险可控,有预案"
4. "我们有多家银行资源"
5. "按合同约定,已完成部分不退"
6. "我理解您的顾虑,我们详细聊聊"
7. "每个客户情况不同,我们尊重您的决定"

---

### 附录C: 沟通检查清单

**每次沟通后,自查**:
- [ ] 是否了解了客户的核心需求?
- [ ] 是否给客户留下了专业印象?
- [ ] 是否回答了客户的主要疑问?
- [ ] 是否推进了下一步行动?
- [ ] 是否记录了关键信息?
- [ ] 是否做了沟通记录?
- [ ] 是否设定了下次联系时间?

### 附录D: 场景切换话术

**从筛选到诊断**:
```
"电话里不太好说清楚,我想当面给您做个详细诊断,
大概1小时,您这周X或X方便吗?"
```

**从诊断到签约**:
```
"方案您也看了,疑问也都解答了,
如果确定合作,我们下周就可以启动,
您看这周X或X方便签约吗?"
```

**从签约到执行**:
```
"合同已经签了,接下来我们会:
1. 对接后端(1-2天)
2. 做测试操作(1天)
3. 正式启动(下周)
我会全程跟进,有问题随时联系我。"
```

**从执行到申请**:
```
"XX总,恭喜!开票已经完成,
现在我们可以向银行申请贷款了。
您这边需要准备XX材料,
我们下周一起去银行提交申请。"
```
