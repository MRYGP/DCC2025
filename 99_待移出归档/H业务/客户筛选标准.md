# H业务客户筛选标准 v1.0

**更新日期**：2025-12-07  
**适用阶段**：Phase 0（跑通SOP阶段）  
**目标**：筛选出2-3个首批客户

---

## 一、筛选流程

```
Step 1：列出候选名单
│   你来做：列出5-10个朋友/关系密切的客户
│
Step 2：硬性条件过滤
│   根据下表排除不符合的
│
Step 3：软性条件评分
│   给每个候选客户打分
│
Step 4：沟通意愿
│   你来做：跟候选客户初步沟通
│
Step 5：确定首批客户
    选出2-3个
```

---

## 二、硬性条件（必须全部满足）

| 序号 | 条件 | 要求 | 原因 | 验证方式 |
|------|------|------|------|---------|
| 1 | 企业成立年限 | ≥1年 | 银行准入门槛 | 营业执照 |
| 2 | 企业经营状态 | 正常（非异常/非吊销） | 银行准入门槛 | 企查查/天眼查 |
| 3 | 法人征信 | 无M3+逾期，M2≤2次 | 银行准入门槛 | 个人征信报告 |
| 4 | 企业征信 | 无重大逾期/诉讼 | 银行准入门槛 | 企业征信报告 |
| 5 | 有真实业务 | 非空壳公司 | H业务依托真实经营 | 流水/实地了解 |
| 6 | 行业可做 | 非限制行业（见下表） | 银行行业限制 | 营业执照经营范围 |
| 7 | 法人年龄 | ≤65周岁 | 平安银行要求 | 身份证 |
| 8 | 法人身份 | 法人且占股/前两大股东之一 | 平安银行要求 | 工商信息 |

### 限制行业清单（需要验证 ⚠️）

以下行业**可能**被银行限制，需要进一步确认：

| 行业 | 限制程度 | 备注 |
|------|---------|------|
| 房地产开发 | 可能限制 | 待验证 |
| 金融/类金融 | 可能限制 | 典当、小贷、P2P等 |
| 娱乐/KTV/夜总会 | 可能限制 | 待验证 |
| 采矿/煤炭 | 可能限制 | 待验证 |
| 钢贸 | 部分银行限制 | 需要具体问 |

**你需要验证**：找银行或助贷朋友确认具体哪些行业不能做。

## 三、软性条件（用于评分排序）

| 条件 | 理想情况（5分） | 良好（3分） | 一般（1分） | 不建议（0分） |
|------|----------------|------------|------------|--------------|
| **年营收** | ≥1000万 | 500-1000万 | 200-500万 | <200万 |
| **现有开票额** | ≥200万/年 | 100-200万/年 | 50-100万/年 | <50万/年 |
| **目标贷款额度** | 100-300万 | 50-100万 | 30-50万 | <30万（ROI低） |
| **时间紧迫度** | 不急，可等6个月 | 可等4个月 | 希望3个月内 | 非常急（不适合） |
| **配合度预期** | 高度配合 | 较配合 | 一般 | 配合度低 |
| **关系亲密度** | 铁哥们/家人 | 好朋友 | 普通朋友 | 不太熟 |
| **信任基础** | 完全信任 | 较信任 | 一般 | 信任不足 |

## 四、Phase 0 特殊考量

首批客户不是为了赚钱，而是为了**跑通SOP + 积累案例**。

### 4.1 首批客户理想画像

```
✅ 朋友或关系密切的客户
✅ 企业有真实业务，年营收500万+
✅ 现有开票额偏低（100万以下），有改造空间
✅ 想贷50-100万，但现在贷不到或额度不够
✅ 不急，愿意等3-6个月
✅ 信任你，愿意提供网银盾配合操作
✅ 对结果有合理预期（理解有风险）
```

### 4.2 首批客户不适合的情况

```
❌ 不熟的客户（沟通成本高，容错空间小）
❌ 空壳公司或无真实业务
❌ 非常急需用钱（等不了3-6个月）
❌ 征信有问题
❌ 对结果期望过高（要求100%保证）
❌ 配合度低（不愿意提供网银盾）
```

## 七、沟通话术建议

### 7.1 开场（试探需求）

```
"最近生意怎么样？资金周转有没有压力？"
"你们公司现在有贷款吗？利率多少？额度够不够用？"
```

### 7.2 介绍方案（如果对方有兴趣）

```
"我最近在做一个业务，帮企业提升贷款额度。"
"主要是帮企业把开票做起来，然后申请银行的发票贷。"
"大概需要3-6个月时间，投入十几万，预计能贷到50-150万。"
"你有没有兴趣了解一下？"
```

### 7.3 试探配合度

```
"这个需要你配合提供网银盾，我们来操作贸易流程。"
"资金不过夜，你只需要配合开票就行。"
"你觉得这个能接受吗？"
```

### 7.4 管理预期

```
"这个不是100%保证的，但根据以往案例，成功率挺高的。"
"前期我们先按成本价来做，主要是跑通流程。"
"你觉得怎么样？"
```
