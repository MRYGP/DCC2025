## ✅ 代码/文档质量标准

### 代码质量标准

**Python代码**：
- [ ] 有注释（函数说明、复杂逻辑说明）
- [ ] 有类型提示（Python 3.6+）
- [ ] 有错误处理（try-except）
- [ ] 有单元测试（至少核心函数）
- [ ] 符合PEP 8规范

**示例**：
```python
def calculate_roi(
    cost: float,
    effect: float,
    period: int
) -> float:
    """
    Calculate ROI for transformation.
    
    Args:
        cost: Total cost of transformation
        effect: Expected effect (quota increase)
        period: Period in months
        
    Returns:
        ROI percentage
        
    Raises:
        ValueError: If cost <= 0 or period <= 0
    """
    if cost <= 0 or period <= 0:
        raise ValueError("Cost and period must be positive")
    
    try:
        roi = (effect - cost) / cost * 100
        return round(roi, 2)
    except Exception as e:
        print(f"Error calculating ROI: {e}")
        return 0.0
```
