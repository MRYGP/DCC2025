## 一、DCC2025 当前状态诊断

### 1.1 项目在做什么？

**一句话**：用「贷查查·企业贷款体检」作为免费入口产品，前端诊断+分诊，后端对接贷款/过桥/典当/做量/体质改造等变现业务。

**业务闭环结构**：
```
┌──────────────────────────────────────────────────────────────────┐
│  前端：贷查查（免费体检入口）                                      │
│  • 人工版：7连问微信问诊（5分钟）                                  │
│  • 自动化版：15道题线上问卷（3分钟）                               │
│  → 产出：体检报告 + 体质分级（A/B/C）+ 方案推荐                    │
└──────────────────────────────────────────────────────────────────┘
                              ↓
┌──────────────────────────────────────────────────────────────────┐
│  中台：诊断 & 方案推荐                                            │
│  • 银行政策库：20+产品（发票贷/税贷/高新贷/抵押贷等）              │
│  • 改造方案库：12种方案（开票提升/纳税提升/资质认定/征信修复等）   │
│  • 匹配算法：准入条件判断 → 额度计算 → 缺口识别 → 方案推荐        │
└──────────────────────────────────────────────────────────────────┘
                              ↓
┌──────────────────────────────────────────────────────────────────┐
│  后端：多业务线变现                                               │
│  • 直接银行贷款（税贷/发票贷/流贷/抵押贷）→ 渠道返佣              │
│  • 票据/过桥/摆账/做量 → 资金使用费分成                           │
│  • 典当及其他资金通道 → 合作方返佣                                │
│  • 体质改造项目（H业务等）→ 咨询服务费 + 返佣                     │
└──────────────────────────────────────────────────────────────────┘
```
