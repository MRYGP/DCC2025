## 一、项目背景与核心目标

### 1.1 背景

- 战略定位已经明确：  
  > **我们是“资金体检 + 贷款体质改造”的医生，看病，也开药。  
  > 企业抓不抓药、吃不吃药，由老板自己决定。**
- 前端工具已经有雏形：  
  - 《贷款体检 7 连问完整版 v1.0》（5 分钟快速体检）  
  - 《诊断问卷 v1.0》（15 题深度体检 + 体质评分 + 方案入口）  
- 我的手里有 2 家自己的企业，可以作为「真实样本」，  
  同时有朋友可以基于【法人征信 + 企业征信 + 税务报告】给出一套“外部测评结论”。

### 1.2 本轮实测的核心目标

1. 用 **两家自己的企业** 做「双轨测试」：  
   - 一轨：朋友按他的方式（征信 + 税务报告）做评估；  
   - 另一轨：我按「7 连问 + 诊断问卷 + 体检简报」做评估；  
   - 对比体质判断、额度判断、产品建议的差异。  
2. 用这两家企业，**打磨出一份可复用的《体检简报 + 处方单》模板**。  
3. 在 30 天内，**完成 5–10 家企业的体检**（含自己公司），  
   从中拿到 **1–2 个有体质改造价值的项目雏形**。

## 二、总体策略：医生定位 + 双轨对照

### 2.1 医生定位（统一口径）

- 对外：  
  > 我做的是企业资金体检和贷款体质改造，  
  > 帮你看清：现在能不能贷、值不值得贷、怎么贷更合适。  
  > 看病、开药是我来做，吃不吃药是你自己决定。

- 对内：  
  - 所有动作尽量都归到一条线：  
    > 资金体检 → 体质诊断 → 体质改造方案 → 贷款方案。  
  - 贴现 / 摆账 / 过桥 / 冲存款 / 债券套利 / 税务优化  
    = 「体质改造 & 资金安排」里的工具，不单独对外当主业务宣传。

### 2.2 双轨对照思路

对同一家公司，同时跑两套流程：

1. **外部流程（朋友）**  
   - 输入：法人征信报告 + 企业征信报告 + 税务报告（必要时加流水）。  
   - 输出：  
     - 预估可贷额度区间  
     - 预估利率区间  
     - 推荐产品（税贷 / 票贷 / 抵押 / 流贷等）  
     - 他眼中的“体质评价”（健康 / 一般 / 边缘）  

2. **内部流程（医生体系）**  
   - 步骤：  
     1. 《贷款体检 7 连问》：从贷款存量 / 利率 / 抵押 / 痛点 / 偏好切入，做 5 分钟初筛。  
     2. 《诊断问卷 v1.0》：按字段填完（去年 + 今年开票 / 纳税等），获取体质等级粗判。  
     3. 出一份《资金体检简报》（A4 一页）。  
   - 输出：  
     - 体质等级：A / B / C  
     - 3 个主要优势  
     - 3 个关键问题  
     - 建议路径：现在就贷 / 先做 3–6 个月体质手术 / 暂缓贷款  
     - 粗略产品建议：税贷 / 票贷 / 流贷 / 抵押贷的组合优先级  

3. **对比 & 校正**  
   - 把两套结论放在一张表里，对比：  
     - 体质判断是否接近？  
     - 额度/利率判断谁更保守？  
     - 产品建议是否一致？  
   - 用差异来优化：  
     - 7 连问话术是否需要调整？  
     - 诊断问卷的维度是否足够？  
     - 体质评分逻辑是否要微调？

## 三、时间拆解：今天 / 本周 / 本月

### 3.1 今天要完成的事情（Day 1）

1. **锁定“医生”身份与对外话术**  
   - 调整微信个签 / 朋友圈简介 / 名片简介：  
     > 企业资金体检 & 贷款体质医生｜先体检，再贷款  
   - 自己大声念一遍 3 分钟标准介绍，直到能不看稿说完。

2. **选定两家企业并建立代号**  
   - 公司 A：__________（主营 / 规模 / 行业简述）  
   - 公司 B：__________（主营 / 规模 / 行业简述）

3. **对自己公司跑一遍《7 连问》**  
   - 按《贷款体检7连问完整版v1.0》逐题回答，  
   - 记录在一个简单表格中（建议：一个 Sheet 对应一家公司）。  
   - 写出初步体质判断（例如：A 公司 = B 体质，有发票贷潜力；B 公司 = A 体质，可直接谈额度）。

4. **和朋友对齐测试需求**  
   - 告诉朋友：  
     - 你希望他给出：  
       - 预估可贷额度区间  
       - 预估利率区间  
       - 推荐产品类型  
       - 他对这家企业贷款体质的简短评价  
     - 资料你可以提供：法人征信 + 企业征信 + 税务报告（必要时加流水）。

---

### 3.2 本周要完成的事情（Week 1）

1. **完成两家企业的《诊断问卷 v1.0》填写**  
   - 为公司 A、B 分别建表，按问卷字段逐项填写：  
     - 去年开票额区间 + 今年截至目前开票额区间  
     - 去年纳税额区间 + 今年截至目前纳税额区间  
     - 行业、成立年限、是否有房产、征信情况等。  
   - 注意：先用自己对公司情况的认知填一个「区间级别的估值」，不必每次都翻报告。

2. **制作公司 A、B 的《资金体检简报》（第 1 版）**  
   - 每家公司做一页 A4：  
     - 体质等级：A / B / C  
     - 3 个优势  
     - 3 个问题  
     - 建议路径（现在贷 / 先做体质改造 / 暂缓贷款）  
     - 粗略产品建议（优先税贷 / 票贷 / 流贷 / 抵押）  
   - 保存为：  
     - `案例库/体检简报/公司A-体检简报-v1.0.md`  
     - `案例库/体检简报/公司B-体检简报-v1.0.md`（具体路径后续再定）

3. **把征信 + 税务报告整理给朋友，拿到他的评估结论**  
   - 至少要拿到四个结论：  
     - 可贷额度区间  
     - 利率区间  
     - 推荐产品类型  
     - 他眼中的企业体质评价  
   - 把朋友的结论记录在文档中，方便后续对比。

4. **完成“你 vs 朋友”的第一轮对比分析**  
   - 为公司 A、B 各写一张对比表，包含：  
     - 额度判断差异  
     - 利率判断差异  
     - 产品建议差异  
     - 体质评价差异  
   - 对差异给出简单分析：  
     - 是你问卷信息不足？  
     - 还是你对银行偏好的理解要修正？  
     - 或者朋友更激进/更保守？

5. **从两家企业中，挑选 1 家作为“体质改造潜力股”**  
   - 选择标准：  
     - 老板态度（愿不愿意长期规划）  
     - 企业发展阶段（有无扩张需求）  
     - 基础体质（如果补一补有明显上升空间）  
   - 给这家公司画出一个粗略「3–6 个月体质改造路径」草稿。

### 3.3 本月要完成的事情（未来 30 天）

1. **累计体检 5–10 家企业（含自己公司）**  
   - 来源：  
     - 自己的两家公司（A/B）  
     - 朋友介绍的老板 2–3 个  
     - 合伙人/会计/代账渠道带来的 1–3 个  
   - 对每家企业至少做：  
     - 一次 7 连问（电话/微信）  
     - 一次简版诊断（不一定全部 15 题，可以视情况精简）  
     - 一句体质判断 + 一条建议路径。

2. **从中形成 2–3 个标准化《体检简报 + 处方单》样例**  
   - 选出典型代表：  
     - 1 家 A 体质企业（可以立即做产品配置优化）；  
     - 1 家 B 体质企业（适合做体质手术）；  
     - 1 家 C 体质企业（不建议立刻贷款，先防风险）。  
   - 为每个类型整理：  
     - 一份完整体检简报  
     - 一份体质改造/贷款方案的「处方单」示意。

3. **力争拿到 1–2 个“体质改造项目雏形”**  
   - 目标不是马上赚很多钱，而是：  
     - 验证老板愿不愿意为“体质变好”付费；  
     - 验证 H 业务 + 工具包（贴现/税务优化等）如何组合落地。  
   - 即便项目规模不大，也要完整走完：  
     > 体检 → 体质诊断 → 设计体质改造方案 → 部分执行 → 复查。

4. **版本迭代：根据实战反馈优化 7 连问 & 诊断问卷**  
   - 每隔 1–2 周回顾：  
     - 哪个问题客户最听不懂？  
     - 哪个问题信息量大但难以获取？  
     - 哪个字段是银行端/朋友端特别看重，而问卷里目前没体现的？  
   - 每次迭代只改小地方，确保整体结构稳定。

4. 有至少 1 个“体质改造项目”的明确机会在推进中。
