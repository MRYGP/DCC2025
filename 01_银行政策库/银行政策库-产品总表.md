## 二、核心产品速查表

### 发票贷（3个产品）

| 银行 | 产品名 | 最高额度 | 利率 | 核心条件 | 放款速度 |
|------|-------|---------|------|---------|---------|
| 交通银行 | 发票贷 | 300万 | 3.6%-8% | 成立≥1年，年开票≥100万 | 3-5天 |
| 平安银行 | 发票云贷 | 500万 | 4%-12% | 成立≥1年，年开票≥200万 | 1-3天 |
| 招商银行 | 发票贷 | 300万 | 3.5%-10% | 成立≥1年，年开票≥150万 | 2-5天 |

**发票贷额度算法**：
```
基础额度 = 年开票额 × 5%-30%
影响因素：
+ 开票连续性好：+5%
+ 有其他银行贷款正常还款：+5%
+ 行业优质（制造业等）：+5%
- 开票波动大：-5%
- 行业敏感（房地产等）：-10%
```

| 银行 | 产品名 | 最高额度 | 利率 | 核心条件 | 放款速度 |
|------|-------|---------|------|---------|---------|
| 建设银行 | 云税贷 | 300万 | 3.15%起 | A/B级纳税，年纳税≥2万 | 最快当天 |
| 中国银行 | 银税贷 | 300万 | ≤4.35% | 优质纳税小微企业 | 1-3天 |
| 工商银行 | 税务贷 | 500万 | LPR基础 | 纳税正常，成立≥2年 | 3-5天 |

**税贷额度算法**：
```
基础额度 = 年纳税额 × 5-30倍
影响因素：
+ 纳税等级A级：额度×1.5
+ 纳税等级B级：额度×1.2
+ 连续3年无欠税：+10%
- 纳税等级C/D级：不准入
- 有欠税记录：降级或拒绝
```

**高新企业贷额度算法**：
```
基础额度 = 根据资质等级确定
- 国家级小巨人：最高1000万
- 省级专精特新：最高500万
- 高新技术企业：最高300万
- 科技型中小企业：最高200万

叠加因素：
+ 有股权投资：+投资额50%
+ 有政府补贴：+补贴额2倍
+ 有核心专利：+50-100万
```

| 银行 | 产品名 | 最高额度 | 利率 | 核心条件 | 放款速度 |
|------|-------|---------|------|---------|---------|
| 中国银行 | 抵押贷 | 1000万 | 灵活定价 | 房产抵押，产权清晰 | 1-2周 |
| 招商银行 | 经营抵押贷 | 1000万 | 2.98%起 | 房产抵押，优质客户 | 1-2周 |

**抵押贷额度算法**：
```
基础额度 = 房产评估值 × 抵押率
- 住宅：抵押率70%
- 商铺：抵押率60%
- 厂房：抵押率50%
- 写字楼：抵押率55%

影响因素：
+ 房产位置好（核心区）：抵押率+5%
+ 房产新（10年内）：抵押率+5%
- 房产老（20年以上）：抵押率-10%
- 产权有瑕疵：不准入
```

## 四、产品选择决策树

```
开始
  │
  ├─ 有房产抵押吗？
  │    ├─ 是 → 抵押贷（额度高、利率低）
  │    └─ 否 → 继续判断
  │
  ├─ 法人是特殊人群吗？（大学生/退伍军人）
  │    ├─ 是 → 创业担保贷款（有财政贴息）
  │    └─ 否 → 继续判断
  │
  ├─ 有高新/专精特新资质吗？
  │    ├─ 是 → 专精特新贷/科创贷
  │    └─ 否 → 继续判断
  │
  ├─ 年开票额≥100万吗？
  │    ├─ 是 → 发票贷
  │    └─ 否 → 继续判断
  │
  ├─ 纳税等级A/B级吗？
  │    ├─ 是 → 税贷
  │    └─ 否 → 继续判断
  │
  └─ 有稳定经营流水吗？
       ├─ 是 → 经营快贷/微业贷
       └─ 否 → 建议先改造企业体质
```
