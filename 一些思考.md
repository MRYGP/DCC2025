### 3.2 综合成本对比话术



```
算账给客户看：

假设需要500万资金：

方案A：民间借贷
├─ 利率：年化18%
├─ 一年利息：90万
└─ 结论：贵

├─ 我们收费：55万（一次性）
├─ 银行利息：20万/年（利率4%）
```

## 二、谈判对象：找谁谈？

### 2.1 谈判对象优先级




### 2.2 各层级谈判策略

| -------------- | ------ | ---------------- | -------------- | -------------------------------------------------- |

| **支行长**     | 合作权 | 支行指标、风险   | 用成功案例背书 | "我们已经和X行合作了，每月推X个客户，通过率X%"     |
| **分行普惠部** | 战略权 | 规模、合规、数据 | 正式商务洽谈   | "我们有精准的企业画像系统，可以帮贵行完成普惠指标" |

### 2.3 找人路径



```
路径A：熟人介绍（最佳）

  └─ 服务过的企业主 → 问他在哪家银行贷过款 → 请他介绍客户经理

路径C：直接拜访（备选）
  └─ 以"企业咨询机构"身份 → 约客户经理交流 → 先建立关系

路径D：行业活动（长期）
  └─ 参加银行组织的企业沙龙 → 认识客户经理 → 后续跟进
```

## 三、价值主张：我们能给银行什么？

### 3.1 核心价值公式



```
贷查查对银行的价值 = 精准客户 + 高通过率 + 批量稳定 + 省时省力

传统渠道：
  银行 ← 广撒网 ← 大量不精准客户 → 审批通过率 30% → 浪费资源

贷查查模式：
  银行 ← 精准推荐 ← 贷查查筛选 → 审批通过率 70%+ → 高效完成KPI
```

### 3.2 价值主张话术

| 场景         | 话术                                                         |
| ------------ | ------------------------------------------------------------ |
| **初次介绍** | "我们帮企业做贷款能力诊断，筛选出符合银行条件的客户，再推荐给合适的银行。" |
| **强调价值** | "我们推荐的客户，都是经过诊断、符合贵行产品条件的，通过率会比普通渠道高很多。" |
| **差异化**   | "我们不是传统中介，不是什么客户都推。我们有一套诊断系统，不符合条件的客户，我们不会浪费您的时间。" |
| **解决痛点** | "您不用再花时间筛选，我们帮您做了前置筛选。符合条件的才推给您，不符合的我们会引导他先做准备。" |

### 3.3 对比传统渠道的优势

| 维度           | 传统贷款中介       | 贷查查             | 银行获益   |
| -------------- | ------------------ | ------------------ | ---------- |
| **客户质量**   | 良莠不齐，有单就推 | 诊断筛选，精准匹配 | 通过率提升 |
| **信息完整度** | 客户说什么是什么   | 结构化诊断数据     | 审批更快   |
| **客户预期**   | 可能被过度承诺     | 如实告知，预期合理 | 减少投诉   |
| **持续性**     | 单次交易           | 长期合作           | 稳定来源   |

```
提价值：
"我们这边有个诊断系统，能帮企业提前做个'体检'。符合条件的，我可以推荐给您；
不符合的，我会告诉他先做什么准备。这样您收到的客户，质量应该会好一些。"

试探合作：
"我这边正好有个企业，条件还不错，您看方便帮看一下？如果合适，我就让他联系您。"
```

### 5.2 强调价值话术

**场景：客户经理问"你们和普通中介有什么区别"**



```
"普通中介是有单就推，不管符不符合条件。我们不一样：

1. 我们有诊断系统：先帮客户做'体检'，知道他能贷多少、适合哪家银行
2. 我们做前置筛选：不符合条件的，我们不会推，省得浪费您时间
3. 我们给结构化信息：客户的开票、纳税、征信情况，我们都会整理好

简单说：您收到我们推荐的客户，通过率应该会比普通渠道高不少。"
```

### 5.3 谈返佣话术

**场景：谈到合作返佣比例**



```
探底：
"咱们这边渠道合作是怎么结算的？按成交金额还是按客户数？"

争取更高返佣：
"我理解您这边有标准，但我们推荐的客户质量确实不一样。
我们做过统计，我们推荐的客户，通过率比普通渠道高30%以上。
对您来说，省了大量无效审批的时间，对银行来说，降低了获客成本。
这个价值，是不是可以在返佣上体现一下？"

退一步：
"这样吧，咱们先按X%试一个月。如果客户质量确实好，通过率高，
下个月咱们再谈提高返佣。您看行不行？"
```

### 5.4 处理异议话术

| 异议                       | 回应                                                         |
| -------------------------- | ------------------------------------------------------------ |
| **"我们有自己的渠道"**     | "理解，多一个渠道多一个选择。我们的客户质量您可以先试试，不满意随时停。" |
| **"返佣比例太高"**         | "咱们先按标准来，等跑顺了、客户质量验证了，再谈调整。"       |
| **"客户信息怎么保护"**     | "客户信息只用于本次贷款申请，我们有保密协议，可以签。"       |
| **"你们怎么保证客户质量"** | "我们有诊断系统，不符合条件的不会推。您可以先试10个客户，看通过率。" |
| **"我们不需要外部渠道"**   | "理解您的考虑。不过现在普惠金融指标压力大，多一个高质量渠道总是好的。" |

## 六、谈判准备清单

### 6.1 谈判前准备

| 准备项           | 内容                             | 用途           |
| ---------------- | -------------------------------- | -------------- |
| **银行产品研究** | 该银行的小微产品、准入条件、利率 | 显示专业度     |
| **成功案例**     | 2-3个已成交案例（可匿名）        | 证明能力       |
| **诊断报告样本** | 脱敏的诊断报告示例               | 展示专业度     |
| **合作方案**     | 简单的合作模式说明               | 让对方快速理解 |
| **名片/介绍**    | 正规的公司介绍                   | 建立信任       |

### 6.2 第一次见面目标



```
目标1：了解银行产品细节（最低门槛、额度算法、隐性条件）
目标2：建立初步信任关系
目标3：达成"试推1-2个客户"的意向
目标4：加微信，保持后续联系

不急于：
- 不急于谈返佣比例（先建立信任）
- 不急于签正式协议（先跑通流程）
- 不急于批量推客户（先验证质量）
```

### 6.3 跟进节奏



```
第1周：初次见面 → 了解产品 → 试推1个客户
    ↓
第2-3周：跟进客户审批结果 → 了解反馈
    ↓
第4周：复盘成功/失败原因 → 优化筛选标准 → 再推2-3个客户
    ↓
第2个月：稳定推送 → 谈返佣细节 → 简单合作协议
    ↓
第3个月+：批量合作 → 争取更高返佣 → 数据反馈合作
```

## 七、风险与应对

### 7.1 常见风险

| 风险             | 可能后果                   | 应对策略                   |
| ---------------- | -------------------------- | -------------------------- |
| **客户经理离职** | 关系断裂，需要重新建立     | 同时维护2-3个客户经理关系  |
| **银行政策变化** | 产品下架或条件收紧         | 实时跟踪政策，多家银行合作 |
| **返佣拖欠**     | 现金流压力                 | 合同明确结算周期，按月结算 |
| **客户被截流**   | 银行直接联系客户，绕过我们 | 保持客户关系，提供持续价值 |
| **合规风险**     | 被认定为非法金融中介       | 定位为"咨询服务"，不碰资金 |

### 7.2 合规红线

| ❌ 绝对不能做           | ✅ 可以做                   |
| ---------------------- | -------------------------- |
| 承诺"包下款""100%通过" | 如实告知"预计额度区间"     |
| 代客户签字、伪造材料   | 帮客户整理材料、规范格式   |
| 收取"打点费""关系费"   | 收取咨询服务费、成功后返佣 |
| 声称和银行有"内部关系" | 说明是正规渠道合作         |
| 经手客户贷款资金       | 只做信息撮合，不碰钱       |

## 八、行动计划

### 8.1 第一个月目标

| 周   | 目标                | 具体行动             |
| ---- | ------------------- | -------------------- |
| W1   | 找到2-3个银行联系人 | 通过熟人/客户介绍    |
| W2   | 初次拜访1-2家银行   | 了解产品、建立关系   |
| W3   | 试推2-3个客户       | 验证客户质量         |
| W4   | 复盘、优化          | 根据反馈调整筛选标准 |

### 8.2 目标银行优先级

| 优先级 | 银行类型   | 原因                     | 目标银行           |
| ------ | ---------- | ------------------------ | ------------------ |
| P1     | 股份制银行 | 灵活、返佣高、合作意愿强 | 平安、招商、兴业   |
| P2     | 城商行     | 本地化、关系好谈         | 长沙银行、华融湘江 |
| P3     | 国有大行   | 规模大但流程长、返佣低   | 建行、工行         |

### 8.3 关键成功指标

| 指标           | 目标值 | 验证周期 |
| -------------- | ------ | -------- |
| 推荐客户通过率 | ≥60%   | 每月     |
| 银行合作数量   | ≥3家   | 3个月    |
| 月推荐客户数   | ≥10户  | 3个月后  |
| 平均返佣比例   | ≥0.8%  | 6个月    |

## 九、一句话总结

> **银行合作的本质是：用精准客户换返佣，用高通过率建信任，用稳定供给做长期。**
>
> **谈判策略：先帮客户经理成单 → 用案例建立信任 → 逐步扩大合作规模 → 争取更好条件**
