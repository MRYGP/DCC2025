## 一、宏观逻辑：为什么这个生意能做

### 1.1 大环境判断

```
核心逻辑：
┌─────────────────────────────────────────────────────────────┐
│  中国超发货币占全世界 40%，美国占 20%                        │
│  ↓                                                          │
│  这些钱必须放出去                                            │
│  ↓                                                          │
│  但中国不搞"大水漫灌"（给个人发钱→存银行→没用）            │
│  ↓                                                          │
│  搞"精准投放"→放给企业                                      │
│  ↓                                                          │
│  企业要有"由头"才能拿到钱（开票、纳税、资质...）            │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 时间窗口

- **未来2-3年**：国家放贷趋势不变
- 银行3%-4%的贷款，不去搞是傻子

### 1.3 市场需求

- 企业普遍"贫血"，流动资金不足
- 很多资产变成"死钱"（固定资产等）
- 企业必须从外面"搞血"

## 二、真实案例数据

### 2.1 团队实操数据

| 指标 | 数据 |
|------|------|
| 养户数量 | 5个户（5个伙计，每人养1个） |
| 月度贸易额/开票额 | 500万/户/月 |
| 年度开票额 | 约6000万/户 |
| 自己走的量 | 每年3000-4000万 |

### 2.2 贷款成果

| 指标 | 低利率版 | 高利率版 |
|------|---------|---------|
| 每户贷款额度 | 500-800万 | 1000-1500万 |
| 年化利率 | 3%-4% | 12%-18% |
| 养户周期 | 约1年 | 约1年 |

### 2.3 核心公式 ⭐

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│    可贷额度 = 年开票额 × 30%（低利率3-4%）                  │
│                                                             │
│    举例：                                                   │
│    月开票500万 × 12个月 = 6000万                            │
│    6000万 × 30% = 1800万（理论最高可贷）                    │
│    实际保守贷：500-800万（年化3-4%）                        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**注**：如果不计较利率（接受12%-18%），可贷额度可以更高。

## 三、成本结构详解

### 3.1 月度成本明细（以月开票500万为例）

| 费用项 | 计算方式 | 金额 | 说明 |
|-------|---------|------|------|
| 操作费用 | 500万 × 3‰ | 15,000元 | 3‰已经是很优质的产品 |
| 账面利润 | 500万 × 2‰ | 6,000元 | 留在账上的利润，私下要返还 |
| 印花税 | 500万 × 0.3‰（单边） | 1,500元 | 万三 |
| 增值税 | 约 | 1,600-1,800元 | 13%的一部分 |
| 其他费用 | 税务周旋、招待费等 | 约1,000元 | 预留 |
| **月度合计** | | **约20,000元** | 满打满算 |

### 3.2 年度成本汇总

| 费用项 | 金额 | 说明 |
|-------|------|------|
| 养户基础成本 | 24万/年 | 2万/月 × 12个月 |
| 人员成本（可选） | 约10万/年 | 法人+员工工资，但这钱可回流 |
| **总养户成本** | **24-25万/年** | 不含人员成本 |

### 3.3 关于人员成本的说明

```
人员成本（法人+员工）：
- 可以从账面利润（6000元/月）中出
- 按3500-4000元/人/月，两人约8000元/月
- 年度约10万

但这个钱：
- 如果法人是自己人/原来的人，钱最终回到自己口袋
- 相当于"左手倒右手"
- 所以实际成本不算这部分
```

## 四、综合成本计算

### 4.1 完整成本模型

以**贷款500万、年化4%**为例：

| 项目 | 金额 | 说明 |
|------|------|------|
| 养户成本 | 25万/年 | 做得很完善的成本 |
| 贷款利息 | 20万/年 | 500万 × 4% |
| **综合成本** | **45万/年** | |
| **等效年化** | **9%** | 45万 ÷ 500万 |

### 4.2 成本对比

| 融资方式 | 等效年化成本 | 说明 |
|---------|-------------|------|
| H业务+银行贷款 | **9%** | 养户25万 + 利息4% |
| 民间借贷 | 15%-24% | 而且额度不一定有 |
| 小贷公司 | 12%-18% | 而且额度有限 |

### 4.3 结论

```
年化9%的综合成本，高吗？

→ 不高！

因为：
1. 比民间借贷低很多（15%-24%）
2. 额度大（500-800万）
3. 第二年继续养，成本结构不变
4. 银行贷款稳定、可续贷
```

## 五、第二年及以后

- 一般情况下**继续养户**
- 成本结构类似

```
第二年起：
- 继续养户：25万/年
- 继续享有：500万+贷款额度
- 可能增加：额度提升
```
